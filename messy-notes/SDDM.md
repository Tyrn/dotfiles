# SDDM, or Simple Desktop Display Manager

_Login agent for KDE Plasma, etc. Not recommended for GNOME_

## Themes

- [sddm-eucalyptus-drop](https://gitlab.com/Matt.Jolly/sddm-eucalyptus-drop/)
  - Add your favorite wallpaper to `/usr/share/sddm/themes/eucalyptus-drop/Backgrounds`
  - Modify `/usr/share/sddm/themes/eucalyptus-drop/theme.conf`
    - `BlurRadius="0"`
    - `ForceHideCompletePassword="true"`
