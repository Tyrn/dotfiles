# GTK notes

## Configuration

- Bookmarks on file managers and dialogs. If ruined:

```
mkdir -p ~/.config/gtk-3.0
```

```
touch ~/.config/gtk-3.0/bookmarks
```

```
chown -R $USER:$USER ~/.config/gtk-3.0
```

```
mkdir -p ~/.config/gtk-4.0
```

```
touch ~/.config/gtk-4.0/bookmarks
```

```
chown -R $USER:$USER ~/.config/gtk-4.0
```
